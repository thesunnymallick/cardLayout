
# card Layout
This is react web application. fetch the data from api and show data.





## Tools/Libraries


- React
- React Hooks
- Scss
- react-router-dom

## Live Deployment

https://cardlayout.vercel.app/


## Run Locally

Install dependencies

```bash
  npm install
```

Start the Locally

```bash
  npm start
```
